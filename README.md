# gAcss (gAccess) Audit Tool

**gAcss** — это консольный инструмент для проведения аудита безопасности и анализа структуры файлов в Google Drive. Приложение позволяет быстро просканировать содержимое облачного хранилища, отфильтровать права доступа и выгрузить подробные отчеты в формате Excel.

---

## Основные возможности

* **Полный сканирование:** получение данных о всех файлах на диске.
* **Рекурсивный аудит папок:** анализ содержимого конкретных папок по ссылке или ID.
* **Фильтрация по участникам:** поиск файлов, к которым имеет доступ конкретный пользователь (по Email).
* **Экспорт в XLSX:** формирование подробных таблиц с именами, ссылками, ролями доступа, владельцами и размерами файлов.
* **Безопасный выход:** возможность мгновенного сброса авторизации и смены аккаунта.

## Технологический стек

* **.NET 8.0** (C#)
* **Google Drive API v3** (Авторизация через OAuth 2.0)
* **Spectre.Console** (Профессиональный CLI интерфейс)
* **ClosedXML** (Генерация отчетов Excel)
* **Microsoft Extensions Hosting** (Dependency Injection и архитектура приложения)
