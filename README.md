# gAcss (gAccess) Audit Tool

**gAcss** — консольный инструмент для аудита безопасности и анализа структуры файлов в Google Drive. Позволяет сканировать облачное хранилище, проверять права доступа и выгружать отчеты в Excel.

## Основные возможности

* **Полное сканирование:** получение данных о всех файлах на диске.
* **Рекурсивный аудит:** анализ содержимого конкретных папок по ссылке или ID.
* **Поиск по участникам:** фильтрация файлов, к которым имеет доступ конкретный Email.
* **Экспорт в XLSX:** формирование таблиц с ролями доступа, ссылками и размерами файлов.
* **Управление сессиями:** быстрая очистка токенов командой `logout`.

## Быстрый старт

Для работы приложения требуется файл ключей `credentials.json`. 

<details>
<summary><b>Нажмите здесь, чтобы развернуть инструкцию по получению ключа</b></summary>

### 1. Создание проекта
* Перейдите в [Google Cloud Console](https://console.cloud.google.com/).
* Нажмите **"New Project"**, введите имя (например, `gAcss-Audit`) и нажмите **"Create"**.

### 2. Включение API
* В левом меню выберите **"APIs & Services"** > **"Library"**.
* Найдите **"Google Drive API"** и нажмите **"Enable"**.

### 3. Настройка экрана согласия
* Перейдите в **"OAuth consent screen"**, выберите **"External"** > **"Create"**.
* Заполните обязательные поля (имя приложения и ваш email).
* В разделе **"Test users"** обязательно добавьте свой Email.

### 4. Создание ключа
* Перейдите в раздел **"Credentials"**.
* Нажмите **"+ Create Credentials"** > **"OAuth client ID"**.
* Тип приложения: **"Desktop App"**.
* Скачайте полученный JSON-файл.

### 5. Установка
* Переименуйте скачанный файл в `credentials.json`.
* Положите его в папку `data/` рядом с `gAcss.exe`.

> **Важно:** При первом входе нажмите **Advanced** -> **Go to gAcss (unsafe)**. Это ваше личное приложение, поэтому Google выводит такое предупреждение.
</details>

## Список команд

| Команда | Описание |
| :--- | :--- |
| `v disk` | Просмотр всех файлов в консоли |
| `v folder <url>` | Анализ конкретной папки |
| `v email <email>` | Фильтр по адресу участника |
| `xlsx disk <путь>` | Выгрузка всего диска в Excel |
| `logout` | Сброс авторизации (смена аккаунта) |
| `help` | Показать все доступные команды |
